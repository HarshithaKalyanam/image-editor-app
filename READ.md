# Image Editor App

## Description
This is an advanced image editing web app built using Streamlit and OpenCV.

## Features
- Upload image
- Blur, Sharpness, Brightness, Contrast
- Edge Detection
- Grayscale
- Rotate & Resize
- Flip Image
- Sepia Filter
- Cartoon Effect
- Add Text
- Download Image

## Technologies
- Python
- Streamlit
- OpenCV
- NumPy
- PIL

## How to Run
pip install -r requirements.txt
streamlit run app.py
