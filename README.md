#  Image Editor App

##  Description
This is an advanced Image Editing Web Application built using Streamlit and OpenCV.  
Users can upload an image and apply various filters and transformations.



##  Features
- Upload Image
- Blur, Sharpness, Brightness, Contrast
- Edge Detection
- Grayscale Conversion
- Rotate & Resize Image
- Flip (Horizontal & Vertical)
- Sepia Filter
- Cartoon Effect
- Add Text on Image
- Download Edited Image



##  Technologies Used
- Python
- Streamlit
- OpenCV
- NumPy
- Pillow (PIL)



##  How to Run
1. Install dependencies:
   pip install -r requirements.txt

2. Run the app:
   streamlit run app.py
